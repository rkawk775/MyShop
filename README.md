### 🎠 마이샵 
---
💻 개발 언어 : mysql, java <br>
📜 프로젝트 설명 : <br> <br>
캐치테이블과 유사한 **음식점 예약 및 고객 관리 시스템** 을 개발하였다. <br> 
예약 관리, 고객 상담, 메뉴 관리 기능을 통합하여 매장 운영 효율성을 높이는 구조로 설계하였다.

**예약 기능** 은 관리자 측에서 전체 예약 현황을 체계적으로 관리할 수 있도록 구성하였다. 고객의 예약 정보를 기반으로 조회, 변경, 삭제가 가능하다.

**고객 상담 기능(채팅)** 을 통해 실시간 문의 응대가 가능하도록 하였다.
